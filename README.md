# 📋 Gerenciador de Tarefas

Sistema simples para gerenciamento de tarefas do dia a dia, desenvolvido com **HTML, CSS e JavaScript puro**.

---

## 🚀 Sobre o Projeto

Este projeto foi criado com o objetivo de praticar conceitos de **Front-End**, permitindo que o usuário:

* Cadastre tarefas
* Visualize tarefas
* Edite e exclua tarefas
* Marque tarefas como concluídas

Os dados são armazenados no próprio navegador utilizando **LocalStorage**.

---

## 🎯 Funcionalidades

* ✅ Cadastro de tarefas
* ✅ Listagem de tarefas
* ✅ Busca por título
* ✅ Marcar como concluída
* ✅ Editar tarefa
* ✅ Excluir tarefa
* ✅ Validação de campos
* ✅ Persistência com LocalStorage

## ⚙️ Como Funciona

### 📥 Cadastro

Preencha o formulário e clique em **Adicionar Tarefa**.

---

### 🔍 Busca

Digite no campo de busca para filtrar tarefas pelo título.

---

### ✏️ Editar

Clique em **Editar** para alterar o título ou descrição.

---

### ❌ Excluir

Clique em **Excluir** para remover uma tarefa.

---

### ✅ Concluir

Clique em **Concluir** para marcar a tarefa como finalizada.

---

## 💾 Armazenamento

As tarefas são salvas utilizando o **LocalStorage**, ou seja:

* Os dados ficam salvos no navegador
* Não é necessário banco de dados
* As tarefas continuam após atualizar a página

---

## 📱 Responsividade

O sistema funciona em:

* 📱 Celulares
* 📟 Tablets
* 🖥️ Desktops

---

## ⚠️ Limitações

* Dados salvos apenas no navegador
* Não possui login de usuário
* Não sincroniza entre dispositivos
